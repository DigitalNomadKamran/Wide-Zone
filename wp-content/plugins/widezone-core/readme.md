# Wide Zone Core

Core functionality plugin providing content models, portals, and integrations for the Wide Zone Co. digital platform.

## Requirements
- WordPress 6.4+
- PHP 8.0+

## Features
- Registers missing custom post types, taxonomies, and metadata for industries, projects, jobs, offices, and resources.
- Provides secure REST API endpoints, shortcodes, and dynamic blocks for editorial workflows.
- Adds role-based access controls, settings screens, and email templates for applicant handling.
- Extends sitemaps, SEO tags, and WP-CLI commands for seeding demo content.
